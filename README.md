# aws-lambda-sample

* pyenv
* direnv
* nodebrew(Node.js 4.3.2)

```
$ pyenv install 3.5.2
$ pip install --upgrade pip
$ pip install --upgrade virtualenv
$ virtualenv env
$ source env/bin/activate
$ pip install -r requirements.txt
```
